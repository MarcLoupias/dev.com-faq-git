# FAQ Git pour developpez.com

## Généralités

### Qu'est ce que Git ?

Git est un système de contrôle de révision décentralisé et rapide.

Il a été écrit en 2005 à l'initiative de Linus Torvalds pour gérer les sources du kernel Linux.
