# FAQ Git pour developpez.com

## 11 Les tags

### Comment supprimer une tag ?

```bash
git tag -d v1.4.0
```

La suppression est seulement locale.

Supprimer un *tag* rendu public est une **très** mauvaise pratique. La bonne pratique est de créer un nouveau *tag* !
