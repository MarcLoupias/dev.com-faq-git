# FAQ Git pour developpez.com

## Les dépôts distants (*remotes*)

### Comment ajouter un dépôt distant ?

```bash
git remote add <alias> <chemin/url>
```

Où `<alias>` désigne le nom du dépôt distant (*remote*).
